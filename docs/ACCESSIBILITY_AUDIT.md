# Accessibility Audit Report — CivicPulse RC1

## WCAG 2.1 AA Compliance Checklist
- **Keyboard Navigation**: All buttons, inputs, and modals focusable via `Tab` / `Shift+Tab`.
- **Screen Reader Support**: ARIA tags (`aria-expanded`, `aria-label`, `role="dialog"`) on interactive elements.
- **Color Contrast**: Accessible contrast ratios verified on dark and light themes.
