# India AI Impact Festival 2026: 8-Week Execution Sprint

This document registers the week-by-week development milestones leading to the festival submission.

---

## 1. Weekly Deliverables Matrix

### Week 1: Government Pilot Commitment
*   **Goal:** Secure 1 written commitment/MOU from a local municipality (Pune, Bangalore, or Mumbai).
*   **Success Metric:** MOU signed and posted to professional networks for social proof.

### Week 2: WhatsApp Channel Adapter
*   **Goal:** Build Twilio/WhatsApp webhook receiver inside the FastAPI app.
*   **Success Metric:** 50 users successfully submit photos and receive classified status returns via WhatsApp.

### Week 3: Multilingual Voice Intake
*   **Goal:** Set up Gemini voice-to-text transcript routing for Hindi, Tamil, Telugu, Kannada, and Marathi.
*   **Success Metric:** Generate RTI draft dispatches in the user's native language.

### Week 4: Accessibility Remediations & Bias Auditing
*   **Goal:** Achieve WCAG 2.2 AA compliance and set up automated demographic bias check scripts.
*   **Success Metric:** Lighthouse accessibility score reaches $\ge 90$ across all public pages.

### Week 5: CPGRAMS & Hotspot Predictions
*   **Goal:** Integrate mock CPGRAMS status updates and build spatial DBSCAN hotspot predictions for government dashboards.
*   **Success Metric:** GIS map correctly groups and displays predictions under high density thresholds.

### Week 6: Anti-Abuse & Reputation Score pipelines
*   **Goal:** Set up duplicate image hash checking and user reputation calculators.
*   **Success Metric:** Malicious or duplicate reports are flagged for review without triggering down-stream Gemini costs.

### Week 7: Visual Polish & Documentation Review
*   **Goal:** Align CSS variables with visual token specs. Finalize all pitch slide decks.

### Week 8: Package Submission & Video Demo
*   **Goal:** Record the 2-minute demo video showing end-to-end evidence intake to government reply.
*   **Success Metric:** Submission finalized on the official Intel/MeitY portal.
